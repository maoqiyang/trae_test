# 炫酷贪吃蛇游戏

一个视觉效果炫酷、交互体验流畅的网页版贪吃蛇游戏，使用HTML5 Canvas和JavaScript开发。

## 🎮 游戏特性

- **炫酷视觉效果**：霓虹风格UI、渐变色彩、粒子特效、平滑动画
- **多种控制方式**：键盘方向键、屏幕虚拟按键、触摸屏滑动
- **渐进式难度**：随着分数增加，游戏速度逐渐提升
- **完整游戏系统**：计分系统、等级系统、最高分记录、暂停/继续功能
- **响应式设计**：适配桌面端和移动端设备
- **音效反馈**：吃到食物、升级、游戏结束等场景的音效提示
- **丰富的特效**：食物出现特效、得分特效、等级提升特效、死亡爆炸特效

## 🚀 如何开始

1. 克隆或下载本项目
2. 直接在浏览器中打开 `index.html` 文件即可开始游戏
3. 也可以通过本地服务器运行，例如使用 VS Code 的 Live Server 插件

## 🎯 游戏控制

- **桌面端**：使用键盘方向键（↑↓←→）控制蛇的移动，空格键暂停/继续游戏
- **移动端**：点击屏幕上的虚拟方向键控制，或使用滑动手势控制
- **开始游戏**：点击"开始游戏"按钮或按Enter键
- **重置游戏**：点击"重置"按钮
- **音效开关**：点击"音效"按钮切换音效状态

## 📊 游戏规则

1. 蛇吃到食物后会变长，同时获得分数
2. 撞到墙壁或自己的身体会导致游戏结束
3. 每获得100分升一级，游戏速度会逐渐加快
4. 游戏会记录您的最高分数

## 🛠️ 技术栈

- HTML5 Canvas - 游戏渲染
- JavaScript - 游戏逻辑
- CSS3 - 页面样式和动画
- Web Audio API - 音效播放

## 🎨 游戏特色

- 蛇头有眼睛，并且会随着移动方向改变
- 蛇身体有渐变色彩效果
- 食物有脉动发光效果
- 吃到食物时有得分动画和音效
- 升级时有庆祝特效
- 游戏结束时有爆炸特效

## 🔧 开发说明

游戏使用模块化的代码结构，便于维护和扩展。主要包括以下模块：
- 游戏初始化和配置
- 事件监听和用户输入处理
- 游戏循环和状态更新
- 渲染系统（蛇、食物、粒子效果等）
- 音效系统

## 📱 兼容性

游戏支持现代浏览器，包括Chrome、Firefox、Safari、Edge等。对于移动设备，建议在横屏模式下游玩以获得最佳体验。
